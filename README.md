# 🎛️ サウンドボード

GitHub Pages で動くポン出しサウンドボードです。

## ✨ 機能

- **プリセット音源** — リポジトリの `sounds/` に置いたファイルを全員で共有
- **マイサウンド** — ユーザーが自由に音声を追加（ブラウザのローカルストレージに保存）
- **ドラッグ&ドロップ** でファイルを追加
- 絵文字・カラー選択でボタンをカスタマイズ
- 音量調整・全停止ボタン

---

## 🚀 セットアップ

### 1. リポジトリを準備

```
your-repo/
├── index.html       ← このファイル
├── sounds.json      ← プリセット定義
└── sounds/
    ├── applause.mp3
    ├── airhorn.mp3
    └── ...
```

### 2. GitHub Pages を有効化

1. GitHub リポジトリの **Settings → Pages**
2. Source を `main` ブランチの `/ (root)` に設定
3. Save → 数分後に `https://your-name.github.io/your-repo/` で公開

---

## 🎵 プリセット音源を追加・変更する

### `sounds.json` を編集

```json
[
  {
    "id": "mySound",
    "label": "ボタン名",
    "emoji": "🎵",
    "file": "sounds/myfile.mp3",
    "color": "#f59e0b"
  }
]
```

| フィールド | 説明 |
|---|---|
| `id` | 一意のID（英数字） |
| `label` | ボタンに表示するテキスト（16文字以内推奨） |
| `emoji` | ボタンのアイコン |
| `file` | `sounds/` フォルダからの相対パス |
| `color` | ボタンのアクセントカラー（HEX） |

### 音声ファイルを追加

`sounds/` フォルダに MP3 / WAV / OGG / M4A を置いて `sounds.json` に登録するだけです。

---

## 👤 ユーザーが音声を追加する方法

1. ページ下部の **「＋ サウンドを追加」** パネルを使用
2. 音声ファイルをドラッグ＆ドロップ、またはクリックして選択
3. ボタン名・絵文字・カラーを設定して「追加」
4. ブラウザのローカルストレージに保存されるため、**同じデバイス・ブラウザ**でのみ表示されます

---

## 📝 対応フォーマット

MP3 / WAV / OGG / M4A（ブラウザがサポートするもの）
